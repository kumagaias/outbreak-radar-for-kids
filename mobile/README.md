# Outbreak Radar for Kids - Mobile App

React Native mobile app providing personalized infectious disease risk assessments for parents of young children.

## Features

- Initial profile setup (child age, location)
- Morning routine check (< 10 seconds)
- Daycare notification response (< 3 minutes)
- Regional outbreak map visualization

## Tech Stack

- React Native (Expo) + TypeScript
- Expo Router (file-based routing)
- React Query (data fetching & caching)
- AsyncStorage (local storage)
- AWS Amplify (backend integration)

## Project Structure

```
mobile/
├── app/              # Expo Router pages
├── components/       # Reusable components
├── lib/              # Utilities and context
├── constants/        # App constants
└── assets/           # Images and fonts
```

## AWS Amplify Configuration

The app connects to AWS backend services using `aws-exports.js`:

```typescript
// aws-exports.js (auto-generated by infrastructure)
export default {
  aws_project_region: 'ap-northeast-1',
  aws_cognito_identity_pool_id: 'us-east-1:xxx-xxx',
  aws_appsync_graphqlEndpoint: 'https://xxx.execute-api.ap-northeast-1.amazonaws.com/dev',
  aws_appsync_region: 'ap-northeast-1',
  aws_appsync_authenticationType: 'AWS_IAM'
};
```

**Update Configuration** (after infrastructure changes):
```bash
cd infra/environments/dev
terraform output -json > ../../../terraform-outputs.json
# Update aws-exports.js with new values
cd ../../../mobile && npm start
```

**Authentication Flow**: Cognito Identity Pool → Temporary AWS credentials → IAM-signed API requests → API Gateway → Lambda

See [backend/README.md](../backend/README.md) for backend API details.

## Deployment

### Web Deployment (AWS Amplify)

**Live URL**: https://dev.d10s8fppfqqa4c.amplifyapp.com

Automated deployment triggers on push to `main` with changes in `mobile/**`. Build process: `npm ci` → `npx expo export --platform web` → Deploy to CloudFront (3-5 minutes).

**Quick Deploy**:
```bash
make mobile-deploy  # Test → Export → Push
```

**Manual Deploy**:
```bash
cd mobile && npx expo export --platform web
```

## Troubleshooting

### API Connection Failures
```bash
curl -X POST https://<api-id>.execute-api.ap-northeast-1.amazonaws.com/dev/recommendations/generate \
  -H "Content-Type: application/json" -d '{"ageRange": "2-3", "geographicArea": "Tokyo", "language": "ja"}'
```

### Build Failures
```bash
rm -rf node_modules package-lock.json && npm install
npx tsc --noEmit
npx expo start --clear
```

### AWS Configuration Issues
```bash
cd ../infra/environments/dev && terraform output -json > ../../../terraform-outputs.json
# Update aws-exports.js with new values
cd ../../../mobile && npm start
```

### Cache Issues
```typescript
import AsyncStorage from '@react-native-async-storage/async-storage';
await AsyncStorage.clear();
```

See [backend/README.md](../backend/README.md) for more troubleshooting.
